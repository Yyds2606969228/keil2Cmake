# SVD/外设解码报告模板

## 1. 目标 (Objective)
统一基于 SVD 的寄存器位域解码输出，让“原始值 -> 语义 -> 结论”的链条可复核。

## 2. 输入 (Input)
- 外设基地址
- 32-bit 或更宽的寄存器快照值
- 匹配的 SVD 节点与芯片型号

## 3. 步骤 (Steps)
1. 对齐 SVD 中的 `peripheral`、`register` 与地址偏移。
2. 读取原始寄存器值并按 field 拆解。
3. 标出被置位、清零或异常组合的字段。
4. 说明每个关键 field 与当前现象之间的关系。

## 4. 输出 (Output)
- SVD 解码清单
- 关键字段解释
- 与问题相关的重点位说明

## 5. 风险 (Risks)
- SVD 可能不完整、过旧或与实际芯片不一致。
- 某些保留位或厂商扩展位可能缺失定义。
- 只做字段翻译，不做现象关联，输出价值会很低。

## 6. 示例 (Example)
```text
外设：TIM2
寄存器：CR1 值 0x0001
解码：CEN=1, UDIS=0, URS=0
结论：计数器已启用，当前“无输出”更可能在通道配置层
```
